Project Sparkle

Constructed an extra sparkly fiber optic lighting tree using a 10W LGB RGB. Created circuit schematic, constructed on breadboard, soldered connections, wrote arduino sketch, and strung the fiber optic cables for the display. Later versions could flash lights through music using Fourier transforms.
